## Goals and features

 1. Enables you to query your Prestashop's data via HTTP request.
 2. Only *SELECT*s are performed, so it's not a full webservice.
 3. 100% coverage of DbQuery API
 4. JSON / XML serialization
 5. OpenSSL encryption
